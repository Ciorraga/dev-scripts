# Python scripts

Python command-line utilities for development and small automation tasks.

Each script has its own folder and README.

## Scripts

| Script | Description |
|---|---|
| [`uuid7gen`](uuid7gen/) | Generates UUIDv7 identifiers from the command line. |
| [`uuid7_sql_replacer`](uuid7_sql_replacer/) | Replaces UUID values in simple SQL `INSERT` and `UPDATE` statements with UUIDv7 identifiers. |
| [`update_sql_field_from_csv`](update_sql_field_from_csv/) | Updates a SQL field using values from a CSV file. |

## Requirements

Check each script README for specific requirements and usage instructions.
